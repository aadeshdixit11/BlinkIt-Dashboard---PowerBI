# BlinkIt-Dashboard-PowerBI

## Overview
This project showcases an interactive **Power BI dashboard** built to analyze and visualize BlinkIt’s sales performance data.  
The dashboard provides dynamic insights into key business metrics, enabling data-driven decision-making.

---

## Key Features
- **Sales Analysis**: Track revenue trends, product performance, and geographical insights.  
- **KPIs Monitoring**: Profitability, average ratings, and sales growth metrics at a glance.  
- **ETL with Power Query**: Cleaned, transformed, and modeled raw sales data for accurate reporting.  
- **Interactive Visualizations**: Drop-down slicers, filtered charts, and matrix views for deeper insights.  
- **Data Source**: BlinkIt Sales Data (Excel file) integrated into Power BI for real-time analysis.  

---

## Purpose
The dashboard is designed to help businesses and analysts:
- Identify top-performing products and categories  
- Understand customer purchasing behavior  
- Optimize sales strategies with actionable insights  
- Present data in a clear, professional, and visually appealing format  

---

## Files in Repository
- `Blinkit Dashboard Practice.pbix` → Power BI dashboard file  
- `Blinkit Sales Data.xlsx` → Raw sales dataset  
- `README.md` → Documentation  

---

## How to Use
1. Download the `.pbix` and `.xlsx` files from this repository.  
2. Open the `.pbix` file in **Power BI Desktop**.  
3. Ensure the dataset (`Blinkit Sales Data.xlsx`) is placed in the same directory or update the data source path.  
4. Explore the dashboard using slicers, filters, and interactive charts.
