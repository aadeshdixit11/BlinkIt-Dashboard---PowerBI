# BlinkIt-Dashboard---PowerBI
This project showcases an interactive Power BI dashboard built to analyze and visualize BlinkIt’s sales performance data. The dashboard provides dynamic insights into key business metrics, enabling data-driven decision-making.
🔑 Key Features-
- Sales Analysis: Track revenue trends, product performance, and geographical analysis.
- Key Performance Indicators (KPIs): Monitor profitability, Average ratings, and Sales growth metrics at a glance.
- ETL with Power Query: Cleaned, transformed, and modeled raw sales data for accurate reporting.
- Interactive Visualizations: Drop-down Slicers, filtered Charts and matrix for deeper insights.
- Data Source: BlinkIt Sales Data (Excel file) integrated into Power BI for real-time analysis.
🎯 Purpose:
The dashboard is designed to help businesses and analysts:-
- Identify top-performing products and categories.
- Understand customer purchasing behavior.
- Optimize sales strategies with actionable insights.
- Present data in a clear, professional, and visually appealing format.
